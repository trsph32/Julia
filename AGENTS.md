# AI Workflow Rules

This document outlines the rules and guidelines for interacting with AI agents in this project.

1.  **Clarity is Key:** Provide clear and concise instructions. The more specific you are, the better the AI can understand and execute your request.
2.  **One Task at a Time:** Focus on a single, well-defined task in each prompt. Avoid combining multiple unrelated requests.
3.  **Provide Context:** If the task involves existing code or files, provide the relevant snippets or file names.
4.  **Iterate:** Start with a simple request and gradually build upon it. Review the AI's output and provide feedback for corrections or refinements.
5.  **Verify:** Always review and test the code or changes generated by the AI. Do not assume it is perfect.
6.  **Safety First:** Do not share sensitive information, API keys, or personal data in your prompts.
